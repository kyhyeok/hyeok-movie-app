============== Introduction 시작 ================

node.js를 다운로드 받고 터미널에 node -v로 버전 및 설치여부 확인

npm -v 역시 버전 및 설치여부 확인 npx -v로 버전 및 설치여부 확인

npm은 node.js 다운로드 하고 설치하면 깔리고

npx는 npm install npx -g (-g는 글로벌로 설치한다는 의미)
